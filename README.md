# Forkify 🍴

A recipe search application built with vanilla JavaScript following the MVC architecture pattern.

## Features
- Search over 1,000,000 recipes
- View detailed recipe information
- Adjust serving sizes
- Bookmark favourite recipes
- Upload your own recipes

## Technologies
- Vanilla JavaScript (ES6+)
- SASS
- Parcel bundler
- Forkify API

## How to run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
